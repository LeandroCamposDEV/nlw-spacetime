## 💻 Projeto
Esse é um projeto Web Responsivel de uma cápsula do tempo para exibir memória em uma linha do tempo.

## 🚀 Tecnologias 
Esse projeto foi desenvolvido durante o NLW da Rocketseat com as seguintes tecnologias:

-HTML
-CSS
-GIT E GITHUB
